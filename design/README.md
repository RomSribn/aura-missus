# Design

Visual references — screenshots, UI ideas, mobile designs. A base of inspiration when implementing frontend.

## Suggested structure

```
design/
  SDU-0001-short-name/
    screenshot.png
    notes.md                  ← optional, describes what's good about it
```

## Suggested rules

- **Prefix:** `SDU-NNNN-short-name/` (Design-UI).
- **Folder per idea.** Screenshots + optional `notes.md`.
- **Next number:** `ls | grep ^SDU | sort | tail -1`

Customise or drop this folder entirely if the project does not need it.
