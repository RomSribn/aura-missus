# Decisions

Important choices made during work — technical or non-technical — that live
outside of tasks and features. Things that need to be remembered, referenced, or
could be revisited.

Not a feature. Not a task step. A decision that affects how the system works or
how we work.

**Decisions must be kept up to date.** When context changes, a decision can be
appended with new info or rewritten. Set `Status: superseded` and link the
replacement rather than deleting history.
