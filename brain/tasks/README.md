# Tasks

Concrete work being done right now. A task can be:
- Implementation of a feature (`PF-*`)
- Follow-up on a decision (`PD-*`)
- Standalone work not tied to a feature (refactoring, cleanup, audit)

Each task is a folder with step files tracking progress from start to user
approval.

**Tasks are history.** Once done, they are not edited — they capture what
happened, when, and why. New work = new task.
